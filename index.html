html & css

html stands for:

Hyper
Text
Markup
Language

HTML is the language that we use when writing web pages. This language is made up of different componants, such as:

  • Tags
  • Elements
  • Attributes

Hypertext refers to the links we use inside websites that allow us to navigate between content pages by clicking. Markup language adds tags to plain text document to define meaning & structure of the text.


CSS stands for:

Cascading
Style
Sheets

CSS works by defining rules that control how different HTML elements will be displayed. CSS rules are made up of:

  • Selectors
  • Properties
  • Values

CSS is used to adjust and control the appearance of a website, including the layout, colour, text, elements and other visual aspects.
